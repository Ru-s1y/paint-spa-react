# Pixel Post
前回作成したお絵かきアプリを変更修正した。

## 概要
ドット絵を描いて投稿するアプリ。

ログインしていないくても基本的な機能が使える。

ドット絵を描いて、投稿もしくはローカルにダウンロードできる。

投稿、編集、削除等はログインしていないと使えない。

## 機能
- 投稿閲覧
- ドット絵の作成、保存
- 投稿、編集、削除(要ログイン)
- お気に入り登録
- アルバム登録(お気に入りとは別に管理できる)
- ページネーション機能
- ログイン、新規登録

## 開発環境
- ReactJS(JSフレームワーク)
- Material-UI(デザイン)
- axios(AJAX通信)
